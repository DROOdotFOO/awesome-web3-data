## Analytics & Visualization Tools

### Block Explorers

- [Beaconcha.in](https://beaconcha.in/) - ETH Beacon Chain explorer, insights, and analytics platform.
- [Bitquery](https://bitquery.io/) - Multi-dimensional analytics with tracing for transaction flows.
- [BlobScan](https://blobscan.com/) - Blob transaction explorer (ETH main and testnets).
- [Blockchair](https://blockchair.com/) - Multi-chain explorer supporting Bitcoin, ETH, and 40+ blockchains with advanced search and filtering.
- [Blockscout](https://blockscout.com/) - Open-source blockchain explorer for ETH and EVM-based networks.
- [BscScan](https://bscscan.com/) - Binance Smart Chain block explorer.
- [Etherscan](https://etherscan.io/) - ETH block explorer and analytics platform.
- [mempool.space](https://mempool.space/) - Open-source Bitcoin mempool visualizer and block explorer with REST and WebSocket APIs.
- [Mintscan](https://mintscan.io) - Cosmos ecosystem explorer.
- [PolygonScan](https://polygonscan.com/) - Polygon block explorer.
- [SIM Explorer](https://explorer.sim.io/) - Block explorer for ETH / Base with advanced features.

### Dashboards & Portfolio Trackers

- [DeBank](https://debank.com/) - Multi-chain portfolio tracker with DeFi analytics and protocol breakdowns.
- [Dune Analytics](https://dune.com/) - Community-powered crypto analytics with SQL queries and dashboards.
- [Pharos](https://pharos.watch/) - Stablecoin analytics dashboard tracking supply, peg stability, liquidity, reserves, yield, and risk signals.
- [Rotki](https://rotki.com/) - Privacy-preserving, open-source portfolio tracker and analytics tool.
- [Zapper](https://zapper.xyz/) - Multi-chain dashboard for DeFi portfolio management and yield farming optimization.
- [Zerion](https://zerion.io/) - DeFi portfolio manager with real-time analytics and cross-chain insights.

### Data Visualization & Exploration

- [Allium](https://www.allium.so/) - Accurate, fast, simple blockchain data.
- [Bubble Maps](https://bubblemaps.io/) - Interactive maps for exploring blockchain data.
- [Chainbase](https://chainbase.com/) - SQL-based multi-chain data platform with APIs for building analytics.
- [Cielo Finance](https://cielo.finance/) - On-chain activity tracker and breakout wallet discovery.
- [DefiLlama](https://defillama.com/) - Open DeFi TVL aggregator with customizable charts, metrics, and protocol comparisons.
- [ETH Gas Tracker](https://www.ethgastracker.com/network/mainnet) - Real-time ETH gas prices and historical analytics.
- [Flipside](https://flipsidecrypto.xyz/) - Blockchain analytics on Snowflake SQL with multi-chain support.
- [L2BEAT](https://l2beat.com/scaling/summary) - ETH L2 analytics and risk assessments for rollups and bridges.
- [Map of Zones](https://mapofzones.com/home) - Analytics platform for the Cosmos ecosystem.
- [Token Terminal](https://tokenterminal.com/) - Financial metrics and charts for crypto protocols and applications.

### Market Intelligence & Analysis

- [CryptoPanic](https://cryptopanic.com/) - News aggregator with real-time updates, sentiment analysis, and market impact tracking.
- [DappRadar](https://dappradar.com/) - Dapp discovery and analytics with DeFi tracking and user activity metrics.
- [Glassnode](https://glassnode.com/) - On-chain market intelligence for digital assets.
- [GMGN](https://gmgn.ai/about) - Meme-coin discovery platform. Also, check out [the bots](https://docs.gmgn.ai/index).
- [LlamaRisk](https://portal.llamarisk.com/) - DeFi risk management platform with dashboards.
- [Messari](https://messari.io/) - Market intelligence with research, metrics, and visualization tools.
- [Nansen](https://www.nansen.ai/) - Blockchain analytics with wallet labels and insights.
- [PandaMetrics](https://pandametrics.xyz/) - Blockchain data analytics platform.
- [Rated Network](https://www.rated.network/) - Insights on blockchain networks, focusing on validator performance.
- [Rekt News](https://rekt.news/) - Analysis and reports on DeFi hacks, scams, and exploits.
- [RWA.xyz](https://rwa.xyz/) - Data platform for tokenized real-world assets, stablecoins, and on-chain treasuries.
- [Sentora](https://sentora.com/) - Crypto analytics platform with ML-driven signals.
- [ZeroMEV](https://zeromev.org/) - ETH MEV explorer showing user losses from miner extractable value (MEV).

## Blockchain Oracles

- [API3](https://api3.org/) - First-party oracle solution for smart contracts.
- [Band Protocol](https://bandprotocol.com/) - Cross-chain data oracle for Web3.
- [Blocksense](https://blocksense.network/) - The zk rollup for programmable oracles.
- [Chainlink](https://chain.link/) - Decentralized oracle network for smart contracts with price feeds and off-chain data.
- [DIA](https://diadata.org/) - Open-source, multi-chain oracle platform with customizable data feeds.
- [Pyth Network](https://pyth.network/) - High-frequency, low-latency price feeds from first-party data providers.
- [RedStone](https://redstone.finance/) - Modular oracle with push and pull models for efficient data delivery.
- [SEDA](https://seda.xyz/) - Cross-chain oracle connecting any network with millions of feeds.
- [Tellor](https://tellor.io/) - Decentralized oracle for off-chain data on ETH.
- [UMA](https://umaproject.org/) - Optimistic oracle for DeFi applications.

## Data Providers

- [Covalent](https://www.covalenthq.com/) - Historical datasets, Covalent Unified API: [goldrush API](https://goldrush.dev/).
- [Moralis](https://moralis.io/) - Web3 development platform with comprehensive data APIs for various blockchains.
- [Space and Time](https://spaceandtime.io/) - Decentralized data warehouse with ZK-based Proof-of-SQL for verifiable queries.

## ETL Tools

- [Blockchain ETL](https://github.com/blockchain-etl/blockchain-etl-common) - ETL tools for various blockchains.
- [Cryo](https://github.com/paradigmxyz/cryo) - High-performance blockchain data extraction tool supporting various blockchains.
- [Ethereum ETL](https://github.com/blockchain-etl/ethereum-etl) - Python scripts for ETL jobs on ETH data, including blocks, transactions, and tokens.
- [Squid](https://github.com/subsquid/squid-sdk) - TypeScript ETL toolkit for blockchain data.

## Indexers

- [Envio](https://envio.dev/) - High-performance blockchain indexer with HyperSync engine for multi-chain data.
- [Goldsky](https://goldsky.com/) - Subgraph-compatible indexer with Mirror for streaming blockchain data to your database.
- [goteth](https://github.com/migalabs/goteth) - ETH indexer written in Go, designed for scalability.
- [Indexed](https://www.indexed.xyz/) - Public, decoded crypto datasets. Updated every 10 mins.
- [Kong](https://github.com/yearn/kong) - Real-time and historical EVM indexer.
- [Ponder](https://ponder.sh) - End-to-end type safe EVM indexer based on Viem.
- [Reth Indexer](https://github.com/joshstevens19/reth-indexer) - Indexer which reads directly from the reth database.
- [rindexer](https://rindexer.xyz/) - High-performance ETH indexer written in Rust.
- [Shovel](https://github.com/indexsupply/shovel) - A simple, config-based ETH to Postgres indexer.
- [Subquery](https://subquery.network/) - Multi-chain blockchain data indexer with GraphQL APIs.
- [The Graph](https://thegraph.com/) - Decentralized protocol for indexing and querying blockchain data with GraphQL APIs.
- [TrueBlocks](https://trueblocks.io/) - Open-source ETH indexer and API for detailed blockchain data analysis.

## Infrastructure

- [Censorship.pics](https://censorship.pics/) - ETH censorship dashboard showing statistics for Builders, Relays, and Validators.
- [Checkpoint Sync Endpoints](https://eth-clients.github.io/checkpoint-sync-endpoints/) - Collection of checkpoint sync endpoints for ETH clients.
- [Client Diversity](https://clientdiversity.org/) - Initiative to promote diversity among ETH clients.
- [ETH Nodes](https://ethereumnodes.com/) - Comprehensive list of ETH nodes for developers and researchers.
- [EthSeer](https://ethseer.io/?network=mainnet) - ETH Beacon Chain explorer providing real-time block production information.
- [Etherscan Node Tracker](https://etherscan.io/nodetracker) - Tool for tracking ETH nodes and their performance.
- [EthSunshine](https://ethsunshine.com/) - ETH node monitoring and analytics platform.
- [Lido VANOM](https://app.hex.tech/8dedcd99-17f4-49d8-944e-4857a355b90a/app/3f7d6967-3ef6-4e69-8f7b-d02d903f045b/latest?ref=blog.lido.fi) - [LidoDAO](https://lido.fi/)'s Quarterly Validator Node Operator Monitoring ('VANOM') report.
- [Lucky Staker](https://luckystaker.com/home/) - Platform providing insights and analytics for ETH staking.
- [Mevboost.pics](https://mevboost.pics/) - MEV-Boost dashboard showing mev-boost flow data across Builders, Relays, and Validators.
- [Monitor Eth](https://monitoreth.io/) - ETH monitoring tool providing insights into the consensus layer and ETH blobs.
- [NodeWatch](https://nodewatch.io/) - Monitoring tool for ETH nodes, providing real-time data and alerts.
- [Rollup.wtf](https://rollup.wtf/) - Realtime data for Rollups and Blobs.
- [StakeWise Helm charts](https://github.com/stakewise/helm-charts) - Charts for deploying different infrastructure components on Kubernetes.
- [Timing.pics](https://timing.pics/) - ETH timing dashboard showing timing games and their effect on the consensus layer.

### Load Balancers

- [blutgang](https://github.com/rainshowerLabs/blutgang) - The wd40 of ETH load balancers.
- [dshackle](https://github.com/emeraldpay/dshackle) - Fault tolerant load balancer for ETH and BTC APIs.
- [erpc](https://github.com/erpc/erpc) - Embedded RPC.
- [nexus](https://github.com/whats-good/nexus) - Load balancing blockchain RPC reverse proxy in TypeScript. [Rust implementation](https://github.com/whats-good/rpc-gateway) W.I.P.
- [RouteMesh](https://routeme.sh/) - Intelligent RPC routing and load balancing across multiple providers.

## RPC Providers

- [Alchemy](https://www.alchemy.com/) - Blockchain development platform with APIs, SDKs, and node infrastructure.
- [Ankr](https://www.ankr.com) - Web3 API provider with a free tier. Supports Web3 login and payments for the lower tier plan.
- [Chainstack](https://chainstack.com) - Web3 APIs including IPFS access and Subgraphs hosting.
- [dRPC](https://drpc.org/) - Decentralized RPC network with multi-provider aggregation and flat-rate pricing.
- [Flashbots](https://protect.flashbots.net/) - Flashbots Protect RPC endpoint for MEV-protected ETH transactions.
- [Helius](https://www.helius.dev/) - Solana RPC & API platform providing comprehensive infrastructure, NFT APIs and transaction APIs.
- [Infura](https://infura.io/) - ETH and IPFS APIs and developer tools for scalable dApp infrastructure.
- [Lava Network](https://gateway.lavanet.xyz/chains) - Decentralized RPC provider network supporting various blockchains.
- [QuickNode](https://www.quicknode.com/) - Multi-chain node infrastructure for Web3 developers.
- [Skip Protocol](https://docs.skip.build/connect/developers/providers) - RPC provider offering access to various blockchain networks.
- [Triton One](https://triton.one/) - High-performance RPC solutions for Solana, Sui, and other blockchains with dedicated infrastructure.

## SDKs and Libraries

- [Anchor](https://www.anchor-lang.com/) - Framework for Solana smart contract development in Rust.
- [ethers.js](https://docs.ethers.io/) - Complete ETH library and wallet implementation in JavaScript.
- [Foundry](https://book.getfoundry.sh/) - Modern EVM smart contract development toolchain via CLI and Solidity scripts.
- [Hardhat](https://hardhat.org/) - ETH development environment for professionals.
- [Multiproof](https://github.com/stakewise/multiproof) - Python library for generating Merkle trees and proofs, fully compatible with OpenZeppelin contracts.
- [Tenderly](https://tenderly.co/) - Smart contract debugging, simulation, and real-time monitoring with transaction tracing.
- [thirdweb](https://thirdweb.com/) - Full-stack Web3 SDK with prebuilt contracts, wallet infrastructure, and frontend SDKs.
- [Viem](https://github.com/wevm/viem) - Modern modular TypeScript interface for ETH.
- [Wagmi](https://github.com/wevm/wagmi) - React hooks and utilities for ETH apps. Based on Viem.
- [web3.js](https://web3js.readthedocs.io/) - ETH JavaScript API for interacting with ETH nodes.
- [web3.py](https://web3py.readthedocs.io/) - Python library for interacting with ETH, suitable for data analysis.
